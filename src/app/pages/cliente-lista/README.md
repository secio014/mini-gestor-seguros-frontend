# Mini Gestor de Seguros - Frontend 🛡️

Aplicação frontend moderna desenvolvida para o gerenciamento de clientes e apólices de seguros. Este projeto faz parte do meu foco atual de aprofundamento em tecnologias de alto desempenho para o mercado corporativo.

## 🚀 Tecnologias e Conceitos

* **Angular 19**: Utilização das funcionalidades mais recentes, incluindo componentes *standalone* e configuração via `app.config.ts`.
* **PrimeNG & PrimeFlex**: Interface profissional com componentes de UI responsivos e sistema de layout moderno.
* **RxJS & HttpClient**: Gerenciamento de fluxos assíncronos e consumo de API REST utilizando Observables.
* **TypeScript**: Tipagem forte para garantir a integridade dos modelos de `Cliente` e `Seguro`.

## 🛠️ Funcionalidades

* Listagem dinâmica de clientes e seguros.
* Formulários de cadastro integrados ao backend.
* Navegação SPA com Angular Router.

## 🏁 Como Executar

1.  Instale as dependências: `npm install`
2.  Inicie o projeto: `ng serve`
3.  Acesse: `http://localhost:4200`
